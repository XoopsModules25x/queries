queries - a genealogical query database for XOOPS

Introduction
============
queries is a XOOPS module which implements a query database, in
which researchers can ask and respond to questions related to their
research.

Installation
============
Copy queries directory to the modules directory of your site, then
install like any other module in your site's administration area.

Notes
=====
This module has been tested in XOOPS version 2.5.9.
