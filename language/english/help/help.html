Help: Queries

Description

The queries module implements a query database, in which researchers can ask and respond to questions related to their research.

Permissions

Each group can be assigned these permissions:

  • Auto-Approve - Automatically approve submissions if set, otherwise query submissions must be approved by an administrator before they will be seen.
  • Add/Edit/Delete for Others - If set, members of this group can edit query for other users. This is normally reserved for administrators.
